PROJECT SETUP INSTRUCTIONS

Step 1: Install Required Tools
Go to Company Portal and install:

* Visual Studio Code
* Node.js (Latest)

---

Step 2: Install Git

Open Command Prompt and run:

winget install --id Git.Git -e --source winget

---

Step 3: Goto users folder

C:\Users\san8577
Create a folder PlaywrightRepos

---

Step 4: Restart

Restart your computer (or at least close and reopen Command Prompt)

---

Step 5: Verify Installation

Run the following commands:

node -v
npm -v
git --version

---

Step 6: Clone the Project

Open terminal
type cd C:\Users\san8577\PlaywrightRepos
git clone https://github.com/sandhyauk/javascript.git

npm install
